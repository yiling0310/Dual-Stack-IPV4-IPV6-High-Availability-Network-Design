
# Enterprise Network Security Lab
Dual-stack enterprise network security simulation using Cisco Packet Tracer

## Features
- IPv4/IPv6 Dual Stack
- VLAN Segmentation
- OSPFv2 & OSPFv3
- WPA2-Enterprise
- RADIUS Authentication
- DNS / HTTP / FTP / Email Servers
- ACL Security Configuration
- STP Redundancy

## Tools
- Cisco Packet Tracer
